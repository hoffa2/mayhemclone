To run the game you need a python enterpreter and pygame.
There are two players initially, and player 1 steers with key_up(thrust) , key_right(steer right), key_left(steer left), CTRL-left(shoot) and key_down(thrust backwards)
Player 2 steers with w(thrust), d(steer right), a(steer left), SPACE(shoot) and s(thrust backwards)
